# 参照資料と更新方針

このサイトは G検定の学習補助を目的とした独自教材です。公式資料の代替ではありません。

## 主要参照

- JDLA公式「G検定とは」
  - https://www.jdla.org/certificate/general/
- JDLA公式のG検定シラバス、例題、改訂告知

## コンテンツ方針

- 公式シラバスのカテゴリ名を学習構成の軸にする。
- 公式本文や公式例題の大量転載は行わない。
- 解説、問題文、可視化、採点フィードバックはサイト独自に作成する。
- 法律・倫理・ガバナンス分野は更新頻度が高いため、公開前に最新版を確認する。

# G検定のインタラクティブ学習サイト

GitHub Pagesで公開できる、静的HTMLのG検定学習サイトです。

## G検定学習サイト要件定義

- 最新のG検定シラバスと公式資料を収集する。
- シラバスに基づき学習トピックを体系化する。
- インタラクティブなHTML教材と可視化要件を設計する。
- 挿絵やグラフのサンプルと動的変数仕様を作成する。
- Codex用の要件定義書フォーマットで文書化する。

## サイト構成

- `index.html`: 学習ダッシュボード、学習マップ、図解、ミニテスト、用語集
- `assets/css/styles.css`: レスポンシブデザイン、ライト/ダークテーマ
- `assets/js/app.js`: ナビゲーション、進捗、状態管理
- `assets/js/visualizations.js`: 分布、回帰、分類、ニューラルネットワーク、Attentionの図解
- `assets/js/quizzes.js`: 学習モジュール、独自問題、用語集データ
- `docs/sources.md`: 参照資料と更新方針

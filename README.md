# G検定インタラクティブ学習サイト

G検定のシラバス項目を、用語・内容単位で学べる多ページ静的HTML教材です。

公開URL:

https://masaspc.github.io/Gken/

## 今回の方針

トップページだけで完結する構成ではなく、各章の重要用語や学習内容ごとに独立ページを作成します。

各学習ページには以下を入れます。

- 学習目標
- シラバス対応の重要用語
- 本文解説
- インタラクティブな図表
- 確認問題
- 前後ページへのナビゲーション
- ブラウザ内 `localStorage` による完了記録

## ページ構成

- 10章
- 87学習ページ
- 10章トップページ
- 用語集
- 横断復習
- 参照資料

HTMLページ数は合計101ページです。

## 章構成

- 人工知能とは
- 人工知能をめぐる動向
- 機械学習の概要
- ディープラーニングの概要
- ディープラーニングの要素技術
- ディープラーニングの応用例
- AIの社会実装に向けて
- AIに必要な数理・統計知識
- AIに関する法律と契約
- AI倫理・AIガバナンス

## 主なファイル

- `index.html`: 章別学習マップ
- `chapters/`: 章トップと学習ページ
- `glossary/index.html`: 用語集
- `review/index.html`: 横断復習
- `sources/index.html`: 参照資料
- `content/syllabus-map.json`: 章・ページ・用語の構造データ
- `assets/css/styles.css`: サイト共通CSS
- `assets/js/lesson.js`: 各学習ページの図表、採点、進捗保存
- `scripts/generate-site.js`: 静的HTML生成スクリプト

## インタラクティブ図表

学習内容に応じて、以下の図表を使い分けます。

- 分布グラフ
- 散布図
- 分類境界
- ニューラルネットワーク構造図
- Attentionヒートマップ
- フロー図
- 関係マトリクス
- 比較チャート

## 生成方法

```bash
node scripts/generate-site.js
```

生成後、GitHub Pages は `main` ブランチのルートから静的HTMLとして公開します。

## 参照方針

- JDLA公式のG検定ページとシラバスを学習構成の基準にする。
- 公式本文や公式例題の大量転載はしない。
- 解説、問題文、図表、確認問題は独自教材として作成する。
- 法律・倫理・ガバナンス分野は、官公庁一次資料の改訂に合わせて更新する。
